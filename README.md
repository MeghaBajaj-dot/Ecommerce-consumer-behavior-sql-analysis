# E-Commerce Consumer Behavior Analysis (SQL)

## Project Overview
This project analyzes E-commerce consumer behavior using SQL to generate actionable business insights related to revenue, returns, customer satisfaction, loyalty programs, and marketing effectiveness.

## Business Objectives
- Identify key revenue-driving product categories
- Analyze return behavior and customer satisfaction
- Evaluate loyalty program and advertising effectiveness
- Support data-driven strategic decision-making

## Tools & Skills Used
- SQL (CTEs, Window Functions, CASE Statements)
- Data Aggregation & Segmentation
- Business Insight & Recommendation Development

## Key Insights
- Revenue is concentrated in a subset of product categories
- Return rates do not significantly reduce customer satisfaction
- Loyalty programs improve satisfaction but not immediate spend
- Moderate ad engagement drives the highest average order value

## Deliverables
- SQL analysis report
- Dataset used for analysis
